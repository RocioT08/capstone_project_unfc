Data & Infrastructure Layer
Data Sources
💹 Yahoo Finance API 😨 Fear & Greed Index 🪙 OHLCV — Stocks & Crypto 📡 AWS Market Data

Daily ingestion of price data for stocks (NVDA, etc.) and cryptocurrencies (BTC, BNB). Enriched with sentiment indicators from Fear & Greed index.

Supabase — Database & Auth
🗄️ PostgreSQL 🔄 Real-time Subscriptions 🪝 Supabase SDK 🏷️ Assets Table 📋 Historical Prices 🔑 Row-Level Security

Central data warehouse with deduplication via unique keys. Stores assets, historical prices, and user portfolios. Handles authentication with secure JWT tokens.

Feature Engineering Pipeline
